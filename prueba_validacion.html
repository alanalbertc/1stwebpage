Web 1.0


La Web 1.0 era solo de lectura, el usuario no puede interactuar con el contenido de la página (nada de comentarios, respuestas, citas, etc.). Esto estaba limitado a lo que el WebMaster sube a la página.

Algunas características

  • Las páginas son estáticas, muy pocas veces dinámicas.
  • No se actualizaban tan seguido.
  • No se pueden adherir comentarios.
  • hace uso de marcos.
  • libros de vistas online.


Web 2.0


La Web 2.0 ha logrado ofrecer al usuario gran cantidad de recursos de software gratuito, dándole la posibilidad a éste de publicar información en la web. Permite que los contenidos son también construidos y viralizados por los propios usuarios.

Algunas características

  • Simplifica la usabilidad del sitio web.
  • Ahorra tiempo del usuario.
  • Facilita la publicación, la investigación y la consulta de contenido.
  • Facilita las interacciones.
  • Estandariza los lenguajes.
  • Mejora la interacción con el software y el hardware.



Web 3.0


La web 3.0 es un neologismo que se a utilizado para describir la evolución del uso y la interacción en la red a través de diferentes caminos. En eso se incluye las características siguientes:

Algunas características

  • La transformación de la red en una base de datos.
  • La innovación de las tecnologías en la inteligencia artificial.
  • Se implementa la web semántica, la web geoespacial y la web 3D.



Web 4.0


La web 4.0 propone un nuevo modelo de interacción con el usuario más completo y personalizado, no limitándose simplemente a mostrar información, sino comportándose como un espejo mágico que de soluciones concretas a las necesidades del navegante.

Algunas características

  • Nuevos modelos de interacción con el usuario.
  • Nuevos modelos de comunicación entre máquina-máquina.
  • Comprensión del lenguaje natural(NLU).
  • Uso de información del usuario como pueden ser la geolocalización, sensores y análisis de datos.
